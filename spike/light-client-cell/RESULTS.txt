FIRST compiling, TESTABLE AdvanceCert type script - verified in CKB-VM on a real preview cert.

light_client_advance.rs assembles the proven verifier into ONE verify_certificate() and a real
cell transition:
  - reads input checkpoint cell state (epoch||avk_root||total, 48B) and output cell state;
  - verify_certificate (against the input avk): compute_hash recheck (sha256 parts==signed_message)
    + aggregate BLS (msgp=msg||root, empty DST) + stake lottery (is_lottery_won per index, all
    2330) + batch-Merkle membership (reconstruct avk root) + quorum (unique indices >= k=1944);
  - requires output cell == { epoch+1, next_avk_root, next_total } (the Mithril cert chain).
